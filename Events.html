NCI at Frederick Events Calendar

Challenge:

The NCI at Frederick calendar consisted of a one page list of unfiltered event links that took users to a details page with little visual appeal. The client requested a redesign of the existing calendar, along with the following functionality:

  • Event List and Calendar View (Month, Week, and Day)
  • Event Search
  • Public and Private Event Distinction
  • Event Export to Outlook
  • Extensive Filter Menus
  • User Dashboard
  • User Event Request Submission Form
  • User Event Duplication
  • Recurring Events
  • Administrative Dashboard
  • Administrative Approval Workflow
  • Mobile View

Contribution:

High-Fidelity Wireframes, Icons, Comprehensives, HTML, CSS, and JS Implementation, Email Templates, QA, User Training

Tools:

Pen and Paper, Illustrator, Photoshop, MS Visual Studio, JIRA

Technology:

ASP.NET Webforms with FullCalendar
